# E2E Tests

## Requirements

To run the e2e test you'll need these env variables set:
  1. `EXPERIENCE_LAUNCH_IMS_ORG_ID`
  2. `EXPERIENCE_LAUNCH_API_KEY`
  3. `EXPERIENCE_LAUNCH_ACCESS_TOKEN`

## Run

`npm run e2e`

## Test overview

The tests cover:

1. Malformed ims org id, api key or access token
2. `read` APIs
